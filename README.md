# 🛒 E-Commerce Website (Frontend Project)

A responsive and interactive e-commerce website built using **HTML, CSS, and JavaScript**.  
This project demonstrates frontend development skills including UI design, DOM manipulation, and basic state management.

---

## 🚀 Live Demo
 https://keertiyadav-code.github.io/E-Commerce-Website/


## 🚀 Features

- 🛍️ Product listing page with clean UI  
- 🛒 Add to cart functionality  
- ➕➖ Increase/Decrease product quantity  
- 🧺 Remove items from cart  
- ❤️ Wishlist (add/remove products)  
- 🔍 Search functionality to filter products  
- 📱 Fully responsive design for mobile, tablet, and desktop  
- ⚡ Dynamic UI updates using JavaScript  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- Vanilla JavaScript (ES6)

---

## 💡 Key Concepts Used

- Event Handling  
- Array/Object based state management  
- Dynamic rendering of UI  
- Responsive design principles  
- LocalStorage (if implemented)
